<h2 align="center">🧠 Project Overview</h2>

<p align="center">
<b>JobSpoon</b>은 AI 기반 모의면접을 통해 개발자의 역량을 진단하고,<br/>
출석·퀴즈·스터디 등 다양한 활동 데이터를 종합하여<br/>
<b>개인 성장과 신뢰도를 시각화</b>하는 국내 최초의 AI 개발자 성장 관리 플랫폼입니다.
</p>
